How to use this app:

  • Upload a starting square image and an ending square image (JPG or PNG format)
  • Adjust the morphing duration (default is 3 seconds)
  • Set the output file name (default is morph.mp4)
  • Click "Generate Morph Video" to create the animation
  • The app will normalize both images to the same square dimensions
  • Each pixel will gradually morph from the starting image to the ending image
  • Once complete, you can play the video and download it

Image Morphing App | Uses HTML5 Canvas for pixel-based morphing | No external libraries required
